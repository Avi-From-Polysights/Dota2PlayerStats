# Contributing

Thanks for your interest in Dota 2 Player Stats. This project is open source ([MIT](LICENSE)) and contributions are welcome.

## How to contribute

1. **Check existing [issues](https://github.com/Avi-From-Polysights/Dota2PlayerStats/issues)** — someone may already be working on it, or you can discuss the idea first.
2. **Fork** the repository and create a branch from `master`.
3. **Make your changes** — keep diffs focused; match existing code style.
4. **Test locally** — run a static server (`npx serve .`) and verify Analyze/Tools still work.
5. **Open a pull request** against `master` with a clear description and test notes.

## Pull request reviews

- All pull requests require **approval from the maintainer** ([@Avi-From-Polysights](https://github.com/Avi-From-Polysights)) before merge.
- The maintainer merges approved PRs; please do not merge your own PR unless explicitly asked.
- Feedback may take a few days — ping on the PR if it stalls.

## What we're looking for

- Bug fixes and regression tests
- UX improvements (especially mobile/responsive)
- OpenDota data handling, caching, and parse reliability
- Documentation and issue repro steps

## Development

See [README.md](README.md) for local setup. No build step — static HTML/CSS/JS modules served as-is. GitHub Pages deploys automatically on push to `master`.

## Code of conduct

Be respectful in issues and PRs. Constructive criticism and good-faith collaboration only.
